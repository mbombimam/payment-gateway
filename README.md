# payment-gateway

## Requirements
PHP >=7.1.20 

## Database 
PostgresSQL

## Installation
```
git clone https://github.com/mbombimam/payment-gateway.git
```

## On your own:
1. Import db-payment.sql PG-Admin
2. RUN APP PHP -S localhost:3030
3. Import Insomnia_payments_rest_api.json (https://insomnia.rest/download)
